# Event API

API REST para gerenciamento de eventos.

## 🚀 Tecnologias
- Java 21
- Spring Boot
- PostgreSQL
- Flyway

## 📌 Funcionalidades
- Criar eventos
- Validação de dados
- Persistência em banco

## ▶️ Como rodar
1. Clonar o projeto
2. Configurar o banco
3. Rodar aplicação

## 📬 Endpoints
POST /api/events
